# DHF 抖音视频发布任务

自动调用 DHF Agent 任务，将视频发布到抖音平台。

## 功能特性

- 📹 **视频上传** - 自动上传视频到抖音
- 📝 **内容支持** - 支持填写视频标题和内容文案
- 🎬 **多视频** - 支持一次上传多个视频
- 💾 **草稿模式** - 可选择直接发布或保存为草稿
- 🔄 **自动化** - 浏览器自动完成所有操作

## 快速开始

### 基本用法

```bash
# 发布单个视频
/dhf-douyin-video-publish-task \
  --title "hello" \
  --content "hello!" \
  --video "./video.mp4"
```

### 多个视频

```bash
# 发布多个视频
/dhf-douyin-video-publish-task \
  --title "视频标题" \
  --content "视频内容" \
  --video "./part1.mp4" \
  --video "./part2.mp4"
```

### 保存草稿

```bash
# 不自动发布，保存为草稿
/dhf-douyin-video-publish-task \
  --title "标题" \
  --content "待补充..." \
  --video "./draft.mp4" \
  --autoSend "N"
```

## 参数说明

| 参数 | 简写 | 必填 | 默认值 | 说明 |
|------|------|------|--------|------|
| `--title` | `-t` | ✅ | - | 视频标题 |
| `--content` | `-c` | ✅ | - | 视频内容文案 |
| `--video` | `-v` | ✅ | - | 视频文件路径（可多次使用） |
| `--autoSend` | `-a` | ❌ | Y | 是否自动发布 |
| `--check` | | ❌ | - | 检查连接 |
| `--help` | `-h` | ❌ | - | 显示帮助 |

## 前置条件

### 1. 安装 DHF Agent

```bash
# 检查安装状态
/dhf-install-agent --status
```

### 2. 启动 DHF Agent

```bash
# 启动 DHF Agent
/dhf-install-agent --open
```

### 3. 登录抖音

首次使用前需要登录抖音：
1. 打开浏览器访问 https://douyin.com
2. 登录你的账号
3. 登录状态会被浏览器保存

## 常见问题

### Q: 提示 MCP 服务不可用？

**A:** 确保 DHF Agent 正在运行：
```bash
/dhf-install-agent --open
```

### Q: 需要登录抖音？

**A:** 首次使用需要手动登录一次：
1. 打开浏览器访问 douyin.com
2. 登录你的账号
3. 之后可以自动发布

### Q: 视频文件不存在？

**A:** 检查视频文件路径：
- 确保路径正确
- 使用绝对路径更可靠
- Windows 路径使用双反斜杠或正斜杠

### Q: 视频上传失败？

**A:** 可能的原因：
- 网络连接问题
- 视频文件格式不支持
- 视频文件过大
- 尝试压缩视频后重新上传

### Q: 如何保存草稿？

**A:** 使用 `--autoSend "N"` 参数：
```bash
/dhf-douyin-video-publish-task \
  --title "标题" \
  --content "内容" \
  --video "./video.mp4" \
  --autoSend "N"
```

## 技术细节

- **任务 ID**: `BK7uNW`
- **MCP 服务**: `dhf_rpa_task`
- **执行超时**: 10 分钟
- **支持格式**: mp4, mov, avi 等常见视频格式
